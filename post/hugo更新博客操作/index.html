Hugo新建文档操作

Oct 7, 2020 · 1 min read

hugo更新博客操作

1. 使用 hugo 命令新建一个 md 文件

命令：hugo new FileName.md（myblog目录下）

2. 将 GitHub 与自己的 hugo 主题关联

hugo --theme=soho --baseUrl="https://liam12138.github.io/" --buildDrafts

3. 将博客部署到远程仓库里

  • 切换到 public 目录 cd public
  • 接下来输入以下指令
git add .
git commit -m "yyyy/mm/dd-hh:mm"
git push -u origin master
